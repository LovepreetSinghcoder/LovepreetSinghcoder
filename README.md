### 🚀 Featured Projects

#### 🏆 LeagueX — Multi-Sport Fantasy Platform *(@ Throughbit Technologies)*
Fantasy sports app extended from cricket (IPL) to football (FIFA World Cup 2026).
- Built with **React Native, NestJS, Prisma, PostgreSQL, Redux Toolkit, TypeScript**
- Designed a **Strategy-pattern-based** (`ISportStrategy`) squad validation engine to support multiple sports off one core
- Built a **MMKV cache-first layer** with stale-while-revalidate patterns and Redux `condition()` guards to cut redundant network calls
- Implemented transfer-locking logic for staggered World Cup matchdays and sport-based dynamic theming
- Shipped on the Google Play Store

#### 🚌 OorBus — Crowdsourced Rural Bus Tracker *(Independent)*
Offline-first bus tracking app for rural Tamil Nadu, built to fill a real infrastructure gap.
- **React Native (Expo) + NestJS/Express, Prisma, PostgreSQL**
- Offline route-finding using **Dijkstra's algorithm**, SQLite for local-first data
- Crowdsourced live bus confirmations from riders with a confidence-scoring cron job
- Google OAuth, JWT auth, atomic upserts for consistent crowdsourced writes

#### 🗺️ Whistling Citizen — Civic Reporting App *(@ Throughbit Technologies)*
Learning and building in **Flutter** to extend my mobile skillset beyond React Native.
- Rendering and interacting with **GeoJSON boundary polygons** across all 38 Tamil Nadu districts using `flutter_map` + `latlong2` on OpenStreetMap tiles
- Debugged GeoJSON parsing crashes and built tap hit-testing for polygon selection
- Authored an internal team guide covering the full pipeline from GeoJSON sourcing to interaction

#### 🍔 Shovii — Food Delivery Platform *(Independent)*
Full-stack food delivery ecosystem with three apps: User, Delivery Partner, and Admin.
- **React Native (Expo), Node.js, Firebase**
- Real-time order tracking, JWT-based auth, role-based access
- Deployed and maintained on the Google Play Store

#### 📝 ClzNotes — Blogging Platform
Real-time blogging platform with Firebase Auth and MongoDB-backed content.
- **React.js, Firebase Authentication, MongoDB**
- Scalable REST APIs with a responsive UI
